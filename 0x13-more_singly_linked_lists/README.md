# 0x13. C - More singly linked lists. #
this project will study what a simple linkage is, how it works and its applications. 

## Objective. ##
* At the end of this project, you are expected to be able to explain to anyone, without the help of Google
* Start to look for the right source of information without too much help
* How to use linked lists


## Description. ##


```
this project seeks to identify the structure of a linked list, how it behaves and what function the nodes have in this application. 

the project has 10 exercises

0-print_listint.c
1-listint_len.c
2-add_nodeint.c
3-add_nodeint_end.c
4-free_listint.c
5-free_listint2.c
6-pop_listint.c
7-get_nodeint.c
8-sum_listint.c
9-insert_nodeint.c
10-delete_nodeint.c

```
